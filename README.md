**SkyCast** is a professional, high-performance weather application built in **2024** to provide accurate, real-time weather updates and 5-day forecasts. It features a modern **Glassmorphism UI** that dynamically adapts its aesthetics based on current atmospheric conditions.

## 🚀 Live Demo
> **Check out the project live**: [SkyCast Weather App](https://hareramkushwaha.com.np/SkyCast/)

---

## ✨ Advanced Features

| Feature | Description |
| :--- | :--- |
| **🌍 Real-time Weather** | Instant updates on temperature, condition, humidity, wind, and visibility. |
| **📅 5-Day Forecast** | Comprehensive daily breakdown to plan your week ahead. |
| **📍 Smart Location** | Auto-detects your city with high-accuracy Geolocation API. |
| **🎨 Dynamic UI** | Interface colors and backgrounds change automatically based on weather (Clear, Rain, Snow, etc.). |
| **🌓 Theme Persistence** | Professional Light and Dark modes that remember your preference via `localStorage`. |
| **📱 Mobile First** | Fully optimized for seamless experience on smartphones and tablets. |
| **⚡ Smart History** | Remembers your recently searched cities for instant one-click access. |

---

## 💻 Tech Stack & Architecture

- **Core**: HTML5, CSS3, Vanilla JavaScript (ES6+).
- **Design System**: Premium Glassmorphism UI with custom CSS variables and animations.
- **API**: [OpenWeatherMap API](https://openweathermap.org/api) integration.
- **SEO Ready**: Optimized with semantic tags, Open Graph meta, and `sitemap.xml`.
- **Optimization**: Optimized for performance with minimal asset size and efficient API handling.

## 🛠️ Getting Started

### 1. Prerequisites
- A modern web browser.
- An API key from [OpenWeatherMap](https://home.openweathermap.org/api_keys).

### 2. Installation
```bash
# Clone the repository
git clone https://github.com/ha-re-ram/SkyCast.git

# Navigate to the folder
cd SkyCast

# Open the app
open index.html
```
